# CC Marketplace

A curated bundle of Claude Code plugins for enhanced productivity, skills, and workflows.

## Installation

Add marketplace to Claude Code:

```bash
/plugin marketplace add kenryu42/cc-marketplace
```

## Plugins

### Plan Export

**Description:** Auto-save your Claude Code plans to the project root when you finish planning

**Categories:** Productivity

**Install:**

```bash
/plugin install plan-export@cc-marketplace
```

**What you get:**

- Automatic plan file export on planning completion
- Plans saved to project root for easy reference

**Repository:** https://github.com/kenryu42/claude-code-plan-export

### Safety Net

**Description:** Block destructive git and filesystem commands in Claude Code.

**Categories:** Security

**Install:**

```bash
/plugin install safety-net@cc-marketplace
```

**What you get:**

- Prevent destructive git and filesystem commands

**Repository:** https://github.com/kenryu42/claude-code-safety-net
